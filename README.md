# My Portfolio

A modern single-page portfolio built with React and Bootstrap, featuring smooth section-based navigation, project showcases with demo links, and a clean presentation of my skills and contact information.
The design is fully responsive and optimized for a seamless user experience across devices.

Built with **React** and **Bootstrap**

## Quick Start

```
# Clone the repo
git clone https://github.com/ChethanFrontendDev/My_Portfolioo.git

# Go to the project folder
cd <your-repo>

# Install dependencies
npm install

# Start development server
npm run dev  # or `npm start` / `yarn dev`

```

## Overview

This portfolio is a Single Page Application (SPA) with smooth scrolling to each section on navigation click.
The page structure includes:

- Home – Introduction and short about-me section
- Projects – List of projects with demo videos and buttons for live preview + source code
- Skills – Technologies, tools, and frameworks I work with
- Connect – Contact details including email and GitHub link
